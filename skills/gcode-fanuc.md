# FANUC G-code 编写规约

## 基础程序框架

```
%
O0001 (PART_NAME)
(MATERIAL: 6061-T6, STOCK 100x80x12)
(SETUP: G54 = LOWER-LEFT, TOP FACE = Z0)
G21 G17 G40 G49 G80 G90
G54
(==== OP1: DRILL ====)
M06 T01
G43 H01
S2500 M03 M08
... (具体加工块)
G80
G00 Z25 M09
M05
(==== OP2: CONTOUR ====)
M06 T02
G43 H02
S8000 M03 M08
...
G00 Z25 M09
M05
M30
%
```

**铁律**：
- 第一行 `%` 和最后一行 `%` 是程序起止标识（FANUC 需要）
- `O0001` 是程序号（4 位数字，O 大写字母）
- 注释用 `()`，**不能嵌套括号**
- 每段最好有 N 编号（N10 N20...），但非必须
- 每行单独一条指令

## 必备模态码（开机就要立的）

```
G21  (公制 mm，G20 是英制)
G17  (XY 平面，G18 是 ZX，G19 是 YZ)
G40  (取消刀补)
G49  (取消刀长补偿)
G80  (取消固定循环)
G90  (绝对坐标，G91 是增量)
G54  (工件坐标系 1，常用)
```

## G 代码速查

| 代码 | 名称 | 用法 |
|---|---|---|
| **G00** | 快速定位 | `G00 X10 Y20`（不带 F） |
| **G01** | 直线插补 | `G01 X10 Y20 F300`（必须给 F） |
| **G02** | 顺时针圆弧 | `G02 X10 Y20 R5 F300` 或 `G02 X10 Y20 I0 J5 F300`（IJ 是圆心相对当前点的偏移） |
| **G03** | 逆时针圆弧 | 同 G02 |
| **G04** | 暂停 | `G04 X1.5`（秒）或 `G04 P1500`（毫秒） |
| **G28** | 回参考点 | `G28 G91 Z0`（增量回 Z0 参考点） |
| **G41/G42** | 刀补左/右 | `G41 D01 G01 X0 Y0 F600`（必须配 D） |
| **G43** | 刀长补偿 | `G43 H01 Z25`（H 是刀长偏置号） |
| **G54-G59** | 工件坐标系 | 1-6 个工件偏置 |
| **G80** | 取消循环 | 一定要写，否则下条指令会重复执行循环 |
| **G81** | 钻孔循环 | `G81 X10 Y10 Z-15 R5 F150`（R 是 retract 平面） |
| **G82** | 钻孔加暂停 | `G82 X10 Y10 Z-5 R5 P500 F100`（沉孔/锪平面，P 暂停 ms） |
| **G83** | 深孔啄式钻 | `G83 X10 Y10 Z-30 R5 Q3 F100`（Q 是每次进给量） |
| **G73** | 高速啄钻 | `G73 X10 Y10 Z-30 R5 Q3 F150`（小退刀，铝合金常用） |
| **G84** | 攻丝循环 | `G84 X10 Y10 Z-15 R5 F375`（F = 螺距 × S） |
| **G98/G99** | 循环退到初始/R 平面 | 默认 G99（退到 R） |

## M 代码

| 代码 | 含义 |
|---|---|
| M00 | 程序停（按 cycle start 继续） |
| M01 | 选择停（操作面板"OPTIONAL STOP"亮才停） |
| M03 / M04 / M05 | 主轴 顺/逆/停 |
| M06 | 换刀 |
| M08 / M09 | 切削液 开/关 |
| M30 | 程序结束 + 复位 + 回程序头 |

## 关键示例

### 例 1：钻 4 个 φ8 通孔（工件 12mm 厚）

```
(==== DRILL 4xPHI8 ====)
M06 T03
G43 H03
S2500 M03 M08
G00 X10 Y10
G00 Z25
G99 G81 X10 Y10 Z-15 R5 F150
X40 Y10
X40 Y40
X10 Y40
G80
G00 Z25 M09
M05
```

要点：
- **Z-15** = -(stock_thickness + 3mm 安全穿透量)
- **R5** = 工件上方 5mm 安全平面
- 第一行 G81 给完整参数，后面同 cycle 的孔只给 X Y，FANUC 会复用上次参数
- 循环结束**必须 G80**

### 例 2：铣外形矩形（50×30，深 5mm，外侧加工）

```
(==== CONTOUR 50x30 OUTSIDE ====)
M06 T01  (PHI10 ENDMILL)
G43 H01
S8000 M03 M08
G00 X-15 Y-15
G00 Z5
G01 Z-5 F300         (下刀)
G41 D01 G01 X0 Y0 F600  (启动左刀补，走到起点)
X50 Y0
X50 Y30
X0 Y30
X0 Y-1               (略过起点 1mm，避免接刀痕)
G40 G00 X-15 Y-15    (取消刀补 + 退刀)
G00 Z25 M09
M05
```

要点：
- **G41 = 左刀补**：用于外形轮廓（刀具在轮廓外侧）
- **G42 = 右刀补**：用于型腔轮廓（刀具在轮廓内侧）
- **D01** 是刀具半径偏置号，对应控制器里的 D01 参数
- 启动 G41/G42 时必须有 G01/G00 直线运动，且距离 > 刀具半径
- 取消 G40 时也要有直线运动远离工件

### 例 3：铣槽（8mm 宽，30mm 长，深 5mm）

```
(==== SLOT 8x30 DEPTH 5 ====)
M06 T05  (PHI8 ENDMILL = 槽宽，无刀补铣)
G43 H05
S8000 M03 M08
G00 X35 Y40           (槽中心起点)
G00 Z5
G01 Z-2.5 F200        (分两层下刀，第一层 -2.5)
G01 X65 Y40 F600      (走槽长方向)
G01 Z-5 F200          (深下到 -5)
G01 X35 Y40 F600      (回起点完成第二层)
G00 Z25 M09
M05
```

要点：
- 槽宽 = 刀具直径时**不需要刀补**，直接走中心线
- 深度大于刀具直径 50% 要分层下刀（一般 ap = 0.5×D）
- 软材料铝可以斜下刀，硬材料预钻 + 刀具铣

### 例 4：型腔（pocket）

```
(==== POCKET 40x20 DEPTH 3 ====)
M06 T01  (PHI6 ENDMILL)
G43 H01
S10000 M03 M08
G00 X20 Y10           (型腔中心)
G00 Z5
G01 Z-3 F100          (慢速下刀)
G01 X20 Y15 F600      (向 +Y 走 5mm)
G02 X25 Y15 R2.5      (右下角圆弧)
G01 X25 Y5
G02 X20 Y5 R2.5
G01 X20 Y10
G00 Z25 M09
M05
```

复杂型腔建议**外接 CAM 软件**（Fusion 360 / Mastercam）出 G-code，agent 不要硬写。

## 编程铁则总结

1. **绝对坐标 G90 优先**
2. **每把刀单独换刀块**：`M06 Tn → G43 Hn → S M03 M08 → 加工 → M09 M05`
3. **进刀前 Z 必到安全平面**（一般 Z25）
4. **下刀方式**：
   - 软材料（铝、塑料）可 G01 直接 Z 下刀
   - 硬材料用 **斜下刀**（同时给 X Y Z）或 **预钻孔**
5. **每个固定循环（G81-G89）后必有 G80**
6. **刀补启用 G41/G42 必有 D 号；取消 G40 必有直线退刀**
7. **F 进给和 S 转速按材料 + 刀具查表，不要拍脑袋**
8. **结束 M30**，否则程序停在最后一行不复位

## 附：FANUC 常用变量与宏

```
#1-#33    本地变量（程序内）
#100-#199 公共变量（断电消失）
#500-#999 公共变量（断电保留）
#3000     报警变量（写值会触发报警）
#3006     带提示停止
```

例：参数化打孔
```
#100 = 4   (孔数量)
#101 = 10  (X 步距)
WHILE [#100 GT 0] DO 1
  G81 X[#101 * (4 - #100)] Y10 Z-15 R5 F150
  #100 = #100 - 1
END 1
G80
```

实战除非批产，否则参数化用得少。
