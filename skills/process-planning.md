# 机械加工工艺规划手册

> **此 skill 是从识图（features）到 G-code 的中间桥梁**。
> 输入：drawing-recognition skill 的输出（features + tolerances + technical_requirements）
> 输出：结构化工艺路线 JSON（工序顺序 / 装夹 / 设备 / 刀具 / 加工阶段）
> 下游：gcode-fanuc skill 据此为每道工序写 G-code

## 0. 30 秒决策树

```
拿到识图结果（features + 公差 + 技术要求）
    ↓
Step 1：判断生产类型 → 单件 / 小批 / 大批 / 大量（影响策略）
    ↓
Step 2：分析毛坯 → 铸件 / 锻件 / 型材 / 焊件（影响首道工序）
    ↓
Step 3：选粗基准（毛坯面）→ 第一道工序定位用
    ↓
Step 4：选精基准（已加工面，全程统一）→ 后续工序定位用
    ↓
Step 5：分阶段（粗→半精→精→光整）
    ↓
Step 6：编排工序顺序，5 大原则：
    基准先行 → 先粗后精 → 先面后孔 → 先主后次 → 配套加工
    ↓
Step 7：插入热处理（预备/最终）+ 辅助工序（检验/去毛刺）
    ↓
Step 8：每道工序选机床+刀具+夹具+量具
    ↓
输出 process_plan JSON
```

## 1. 工艺过程的 5 级层次（必须区分）

```
工艺过程
  └── 工序  (一个工人 / 一台机床 / 连续完成的内容)
       └── 安装 (一次装夹完成的部分)
            └── 工位 (一次安装中改变工件位置的部分；多工位夹具用)
                 └── 工步 (加工面+刀具不变下连续完成的部分)
                      └── 走刀 (一次切除一层金属)
```

- **工序划分依据**：机床/工作地点是否改变 + 加工是否连续
- **同一台机床上换刀** → 同一工序内的不同工步
- **工件转到另一台机床** → 新工序

## 2. 生产类型 → 工艺策略

| 生产类型 | 年产量（中型件 15-50kg）| 毛坯精度 | 机床 | 工装 | 工艺文件 |
|---|---|---|---|---|---|
| 单件小批 | ≤200 | 低（手工造型/自由锻）| 通用机床机群式 / 数控 | 通用工装为主 | 工艺过程卡 |
| 中批 | 200~500 | 中（金属模铸造/模锻）| 部分专用 + 流水线 + 数控 | 专用夹具+可调夹具 | 工艺卡，详细 |
| 大批大量 | >500 | 高（机器造型/精锻）| 广泛专机+流水线 | 高效专用工装 | 过程卡+工序卡，最详 |

⚠️ **不知道生产类型时默认按"单件小批"处理**——选通用机床+通用工装+少专用刀具，最稳。

## 3. 毛坯类型分析（影响首道工序与余量）

| 毛坯 | 制造方法 | 余量 | 工艺特点 | 适用零件 |
|---|---|---|---|---|
| **铸件** | 砂铸/金属型/熔模/压铸/低压/离心 | 大（4-15mm 单边）| 组织粗，内部缺陷多，力学性能差，但形状适应性广，便宜 | 箱体、机座、缸体、活塞环、机床床身、支架 |
| **锻件** | 自由锻/模锻/挤压/轧制/冲压 | 中（3-8mm 单边）| 组织细密，力学性能好（强度/抗冲击/抗疲劳）；难做大件复杂内腔 | 光轴、阶梯轴、曲轴、连杆、齿轮坯料 |
| **焊件** | 手弧焊/气保焊/电阻焊 | 不定 | 周期短、利用率高；抗震差、易变形 | 大型箱体、容器、船舶、化工设备 |
| **型材** | 棒料/管料/板/角钢 | 小 | 直接用，成本低，但形状受限 | 简单轴类、方料件、连接件 |

**毛坯选择决策**：
- 形状复杂 → 铸件
- 力学性能要求高 → 锻件
- 大型件 → 自由锻 + 砂铸
- 中小件批量 → 模锻 + 金属模铸造
- 大批量 + 性能要求高 → 模锻 / 精铸 / 精锻 / 冷挤压
- 单件小批 → 木模手工造型 + 自由锻
- 简单轴类台阶差不大 → 棒料（最便宜）
- 简单轴类台阶差大 → 锻件（省料）

## 4. 基准选择（**工艺路线的灵魂**）

### 4.1 五种基准（一定要分清）

```
设计基准   ──→ 设计图样上确定其他点/线/面位置的基准
工艺基准 ──┬─→ 工序基准  (工序图上标尺寸的基准)
           ├─→ 定位基准  (加工时工件在机床/夹具上定位用) ⭐ 这个最关键
           ├─→ 测量基准  (检验时测量用)
           └─→ 装配基准  (装配时确定零件位置用)
```

定位基准又分：**粗基准**（首道工序用毛坯面）/ **精基准**（已加工面）/ **辅助基准**（专门设计的工艺面）

### 4.2 粗基准选择 4 原则（首道工序用，**只能用一次**）

| 原则 | 说明 |
|---|---|
| **原则 1：选不加工表面** | 保证加工面与不加工面之间的位置精度。多个不加工面时选与加工面位置精度要求最高的 |
| **原则 2：选重要表面** | 重要表面要保证加工余量均匀（如导轨面）|
| **原则 3：同方向只能用一次** | 同一坐标方向粗基准只能用一次（重复用会因毛坯误差导致后续位置偏差累积）|
| **原则 4：表面要平整** | 没有浇口、冒口、飞边、缩孔；定位可靠 |

**例**：阶梯轴（毛坯棒料）→ 选外圆为粗基准车端面+打中心孔。

### 4.3 精基准选择 6 原则（后续所有工序用）

| 原则 | 说明 | 应用 |
|---|---|---|
| **基准重合** | 精基准 = 设计基准 | 避免基准不重合误差 |
| **基准统一** | 尽可能用同一组定位基准加工各表面 | 简化夹具，避免基准转换误差。**如车床主轴用中心孔统一基准加工各外圆** |
| **自为基准** | 加工面本身做精基准 | 仅要求余量均匀（如导轨面精磨）|
| **互为基准** | 两个表面交替做基准反复加工 | 高位置精度 / 余量均匀（如主轴轴颈与锥孔互为基准）|
| **定位可靠** | 选大尺寸表面，承力大、稳定 | 防变形 |
| **结构简单** | 夹具结构简单、夹紧可靠 | 工艺经济性 |

⚠️ 上述 6 原则**会矛盾**——抓主要矛盾，灵活运用。

### 5.2 典型表面加工路线

#### 外圆加工路线
| 路线 | IT | Ra | 适用 |
|---|---|---|---|
| 粗车 → 半精车 → 精车 | 7~8 | 0.8~1.6 | 除淬火钢外各种金属 |
| 粗车 → 半精车 → 粗磨 → 精磨 | 6~7 | 0.1~0.4 | 淬火钢/未淬火钢，但**不宜有色金属**|
| 粗车 → 半精车 → 精车 → 金刚石车 | 5~6 | 0.4~0.025 | 主要用于有色金属 |
| 粗车 → 半精车 → 粗磨 → 精磨 → 光整 | ≥5 | 0.025~0.008 | 极高精度 |

#### 孔加工路线
| 路线 | IT | Ra | 适用 |
|---|---|---|---|
| 钻 → 扩 → 粗铰 → 精铰 | 7~8 | 0.63~2.5 | **直径 < 40mm 的中小孔** |
| 粗镗（或钻）→ 半精镗 → 精镗 | 7~9 | 0.63~2.5 | **大孔径 / 高位置精度孔系 / 单件小批 / 有色金属** |
| 钻 → 拉 | 7~9 | 0.32~2.5 | 大批量盘套类零件圆孔/键孔/花键孔 |
| 粗镗 → 半精镗 → 粗磨 → 精磨 | 6~7 | 0.16~1.25 | 中小型淬硬零件孔 |

#### 平面加工路线
- 不淬硬 + 中小件：**粗铣 → 精铣**（IT7、Ra1.6）
- 淬硬：磨削（IT6、Ra0.32）
- 大型件 / 狭长平面：精刨

### 5.3 加工方法选择的 4 个影响因素
1. **工件材料**：淬火钢→磨；有色金属→精车/金刚镗（避免堵砂轮）
2. **工件形状/尺寸**：箱体孔→镗（不宜拉/磨）；φ>60mm 不宜钻扩铰
3. **生产类型**：大批量→高效设备；单件小批→通用设备+标准刀具
4. **现有生产条件**：充分用现有设备，不轻易上专机

## 6. ⭐ 工序顺序 5 大原则

按以下顺序应用（前面优先）：

### 原则 1：基准先行
**先加工基准面，再加工其他面**。精基准应在工艺过程一开始就加工，为后续提供基准。
- 例：轴类先以外圆为粗基准 → 加工中心孔 → 后续以中心孔为精基准

### 原则 2：先粗后精
**所有表面：粗加工 → 半精加工 → 精加工 → 光整**。
- 粗加工切大量余量，精加工保证质量
- 各阶段间有自然时效作用（应力释放）
- 划分阶段的好处：保证质量 / 合理用设备 / 便于安排热处理 / 便于发现毛坯缺陷

### 原则 3：先面后孔
**箱体/支架类零件：先加工平面，再加工平面上的孔**。
- 大平面先加工后可作定位基准
- 平面加工后孔加工时刀具切入更稳，测量也方便

### 原则 4：先主后次
**主要表面（精度高/装配基面）先加工，次要表面（键槽/螺孔）后加工**。
- 但次要表面要在最终精加工之前完成（不能放到最后）

### 原则 5：配套加工
**有些表面的最终精加工放到部装/总装阶段做**，以保证配套精度。
- 例：车床主轴的法兰、止口、平面在主轴装上车床后才精加工

⚠️ **5 原则会冲突**——抓主要矛盾。判断规则：精度要求 > 工艺顺序简化。

## 7. 加工阶段划分

| 阶段 | 主要任务 | 切除量 | 设备 |
|---|---|---|---|
| **粗加工** | 切除毛坯大部分余量，形状/尺寸接近成品 | 大 | 功率大、刚性好、精度低 |
| **半精加工** | 主要表面达到一定精度，留余量为精加工准备；完成次要表面（钻孔/攻螺纹/铣键槽）| 中 | 通用 |
| **精加工** | 全面保证图样要求（尺寸/粗糙度/位置精度）| 小 | 高精度 |
| **光整加工** | 进一步降低粗糙度（**不能纠正形状/位置误差**）| 极小 | 专用（研磨/珩磨/超精）|

**例外**：要求不高 / 余量很小 / 重型零件 → 可一次成形不分阶段。

## 8. 热处理工序安排

```
预备热处理              最终热处理
────────────           ────────────
退火/正火               淬火
时效                    渗碳淬火
调质                    渗氮淬火
```

| 热处理 | 时机 | 目的 |
|---|---|---|
| 退火/正火 | **毛坯生产后、粗加工前** | 消除组织不均，细化晶粒，改善切削性，消除毛坯应力 |
| 时效（一般铸件）| 粗加工前 或 粗加工后 | 消除应力 |
| 时效（精密件 / 丝杠）| 粗加工后、半精加工后、精加工后多次 | 多次释放应力 |
| 调质 | **粗加工后、精加工前** | 提高力学性能 / 为后续表面处理做组织准备 |
| 淬火 | **精加工前**（一般淬火后磨削最终）| 高硬度（脆性增加，应力大，变形） |
| 渗碳淬火 | **半精加工与精加工之间** | 表面高硬度+耐磨，心部强度+韧性 |
| 渗氮 | **尽量靠后**（精加工后甚至最终）| 高硬度/耐磨/抗疲劳/抗蚀，温度低变形小 |

**淬火件标准工艺路线**：
```
下料 → 锻造 → 正火 → 粗加工 →（调质）→ 精加工 → 表面淬火+回火 → 精磨
```

**渗碳件标准工艺路线**：
```
下料 → 锻造 → 正火/退火/调质 → 粗加工/半精加工 → 渗碳 → 淬火+低温回火 → 磨削
```

## 9. 辅助工序

必须包含：**检验**（关键工序前后/各加工阶段之间/最终）、**去毛刺/倒棱**、**清洗**、**防锈/退磁**

- 检验：关键工序前后 + 阶段交接 + 最终
- 重要件可加 X 射线/磁粉探伤/密封性试验
- 研磨/珩磨后必须清洗（防砂粒残留）
- 工序间和入库前必须清洗+涂油

## 10. 机床和工装选择

### 机床
1. 规格匹配工件（大件大机床）
2. 精度匹配工序要求（不必过高）
3. 生产率匹配产量（小批通用，大批专用）
4. **中小批 + 复杂工序 → 数控机床**
5. 兼顾现有生产条件（不轻易上专机）

### 夹具
- 单件小批：通用（卡盘/回转工作台/平口钳）+ 组合夹具
- 大批量：专用夹具

### 刀具
- 优先标准刀具
- 大批量：复合刀具/专用刀具/可转位刀具/陶瓷刀具/玉米铣刀

### 量具
- 单件：通用（卡尺/千分尺/塞规）
- 大批量：专用量具/极限量规/CMM

## 12. 工艺路线编排实例（速查）

### 例 A：阶梯轴（45 钢，棒料毛坯，单件小批）
```
工序 10：下料（锯床）
工序 20：车（CA6140）
  装夹 1：夹大端，车小端面 + 钻小端中心孔 + 粗车小端外圆 + 倒角
  装夹 2：调头夹小端，车大端面 + 钻大端中心孔 + 粗车大端外圆 + 倒角
工序 30：车（CA6140，以两中心孔定位）：精车外圆
工序 40：铣（X5032）：铣键槽
工序 50：检验
```

### 例 B：箱体类零件（HT150，铸件，单件小批）
```
工序 10：检验毛坯（看缩孔/砂眼/飞边）
工序 20：粗铣大平面（先面后孔原则）—— 用毛坯面做粗基准
工序 30：粗铣其他平面（以工序 20 加工面为精基准）
工序 40：时效（消除粗加工应力）
工序 50：半精铣大平面 + 精铣大平面（自为基准 / 留 0.5mm 精磨余量）
工序 60：钻孔/扩孔/铰孔（精度要求 H7 用钻+铰；H8 H9 钻+精扩）
工序 70：粗镗+精镗高精度孔（H7 镗孔，定位用工序 50 大平面）
工序 80：钳工去毛刺
工序 90：清洗 + 防锈
工序 100：终检
```

## 13. 边界与常见错误

| 错误模式 | 起因 | 正解 |
|---|---|---|
| 把所有工序串到一台机床 | 没看生产类型 | 大批量按工序分散到流水线；单件小批可工序集中（用数控）|
| 钻孔放在最后 | 没遵循先粗后精 | 钻孔属粗加工，必须放精加工前 |
| 高精度 H7 孔用 endmill 直接成型 | 不懂经济精度 | H7 必须钻+铰 或 粗镗+精镗 |
| 粗基准用了两次 | 没遵循"同方向只用一次" | 第二次工序必须改用精基准 |
| 精基准每道工序都换 | 没遵循基准统一 | 用一组精基准贯穿到底（如轴用中心孔）|
| 淬火放在精磨之后 | 时机错 | 淬火放精加工前，最终留磨削纠正变形 |
| 时效漏做 | 忽略应力 | 铸件粗加工后必时效 |
| 没分加工阶段 | 一次性切到底 | 余量大必分粗精，避免应力变形累积 |
| 漏检验工序 | 没考虑质量管控 | 关键工序前后必须检验 |
