# 读机械图纸

## 识图先看四件事
1. **标题栏**（右下角）：零件名、材料、比例、投影法、通用公差等级
2. **视图布置**：哪几个视图（主视、俯视、左视、剖视、局部放大）
3. **比例**：1:1 / 1:2 / 2:1 ...（图纸不一定按 1:1 画）
4. **投影法**：第一角法（中国/欧洲常用）vs 第三角法（美国常用）

## 投影法判断
| 投影 | 俯视图位置 | 左视图位置 |
|---|---|---|
| 第一角（GB） | 主视图**下方** | 主视图**右侧** |
| 第三角（ANSI） | 主视图**上方** | 主视图**左侧** |

标题栏通常有符号区分：第一角是个圆锥从右往左；第三角从左往右。

## 尺寸标注语法

| 标注 | 含义 |
|---|---|
| `100` | 线性尺寸（默认 mm） |
| `φ10` 或 `Ø10` | 直径 10 |
| `R5` | 半径 5 |
| `45°` | 角度 45 度 |
| `5` 下方加 `↓` 或写 `深 5` | Z 方向深度 5mm |
| `M10` | 公制粗牙螺纹（默认螺距 1.5） |
| `M10×1.25` | 细牙，螺距 1.25 |
| `4×φ8` | 4 个 φ8 孔 |
| `□20` | 方形 20×20 |
| `C2` | 2×45° 倒角 |
| `⌴φ20 深 5` | 沉孔（counterbore）φ20 深 5 |
| `⌵90° φ12` | 锥形沉头（countersink）90° 大端 φ12 |

## 公差读法

### 通用公差
标题栏注明：
- "未注公差按 GB/T 1804-m"（中等级）
- "ISO 2768-mK"（中等线性 + 中等角度）

### 配合公差
孔轴配合写在尺寸后，例：`φ20H7`
- 大写字母 = 孔（H/G/F/J...）
- 小写字母 = 轴（h/g/f/k...）
- 数字 = IT 等级（精度，越小越精）

| 标注 | 含义 |
|---|---|
| `φ20H7` | 孔 φ20，上偏差 +0.021，下偏差 0（IT7 等级） |
| `φ20g6` | 轴 φ20，上偏差 -0.007，下偏差 -0.020 |
| `φ20H7/g6` | 孔轴间隙配合 |
| `φ20H7/k6` | 过渡配合 |
| `φ20H7/p6` | 过盈配合 |

**实操规则**：
- H7 及以下精度孔，**铣不动，要铰或镗**
- H8 / H9 高速钢钻头能保证
- H10 普通钻头

### 几何公差（形位公差）
方框语法：`[符号 | 数值 | 基准]`
- `⊥ 0.05 A` — 垂直度 0.05，基准 A
- `∥ 0.02 B` — 平行度 0.02
- `⌀ 0.01 A` — 圆度 0.01
- `⌖ φ0.05 A B C` — 位置度 φ0.05，基准 A B C
- 字母后画三角形或圆 = 基准（如 `[A]`）

## 表面粗糙度

`√` 符号（旧 GB）/ `∇`（车间）/ ISO 写法：
- `Ra 3.2` — 算术平均偏差 3.2 μm（机加工常见）
- `Ra 1.6` — 较细
- `Ra 0.8` — 精磨
- 不标 = 标题栏的"未注表面粗糙度"

## 隐藏线 / 中心线

| 线型 | 含义 |
|---|---|
| 粗实线 | 可见轮廓 |
| 细虚线 | 不可见轮廓（被遮挡） |
| 细点划线 | 中心线 / 对称轴 |
| 双点划线 | 假想线 / 极限位置 |
| 细实线 | 尺寸线 / 引出线 / 剖面线 |

## 剖视图
- 全剖：整个零件剖开
- 半剖：对称件剖一半显示一半
- 局部剖：只剖一个特征（用波浪线分隔）
- 标注："A-A" 表示沿 A-A 截面剖

## 常见陷阱

1. **比例不对应**：图上量出来的距离 ≠ 实际尺寸（除非 1:1）。**永远以标注尺寸为准**。
2. **隐藏特征**：复杂零件主视图看不到的特征要看俯视/侧视/剖视
3. **重复尺寸**：同一尺寸标在不同视图，找最清晰的那个
4. **基准链**：尺寸链一定要顺着基准走，避免累积误差
5. **左右对称件**：图纸经常只画一半 + 对称符号 `─CL─`

## 与 PDF/vision 配合的实操

读 `parsed/page_<N>.png` 时：
- 先看整张图的全貌（视图布置、比例）
- 再放大局部读细节（如果图纸大可以让 parse_pdf 切片）
- `parsed/text.json` 里的文本块带边界框，可以根据**位置**判断这个数字属于哪个尺寸（数字最近的几何线就是它标的）

不要逐字 OCR——视觉模型直接看图比文字更准。
