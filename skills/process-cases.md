# 工艺过程卡案例库

> **使用方式**：写工艺路线遇到下述情形之一时，找最相似案例对照：
> - 多面/多孔零件（→ 案例 1）
> - 多种螺纹规格混合 / 斜孔 (→ 案例 2)
> - 数控车铣联用 / 高精度孔（H7/H8 铰孔）（→ 案例 3）
> - **淬火件 / 合金钢 / HRC > 28**（→ 案例 4，**必读**）
> - **超精度配合（IT6 / 圆柱度 ≤ 0.005）/ 长导轨平面**（→ 案例 5，**必读**）

## 案例 1：箱体（HT200 铸件，成批生产）

**关键参数**：基准 R35 毛坯外圆 + φ30 凸台 + 67×39 凸台高度

```
工序 1：铸（铸造，清理）
工序 2：热处理（时效，消除铸造应力）
工序 3：涂装（非加工外表面涂底漆，防止铸件锈蚀影响装配检查）
工序 4：划线（基准定位用，给后续粗加工提供参考线）
工序 5：车（四爪，分两次装夹粗精车 φ58/φ36 内孔 + 端面，精车留 0.5mm 余量）
工序 6：车（调头粗精车上端面，保证总高 56）
工序 7：⭐ 数控铣（回转台 + 专用铣夹具，4 个工位）
  工位 1（A 向正对主轴）：钻扩铰 φ12 / φ16 / 4×M6 攻丝 / φ8 钻
  工位 2（回转台逆转 90°）：粗精铣 A 面（保证 82）+ 4×M6 攻丝 + φ8
  工位 3（回转台逆转 180°）：钻 φ8
  工位 4（回转台逆转 270°）：钻 φ8
工序 8：钻（基准 φ36 孔，用钻夹具，4×M6 攻丝）
工序 9：涂装（内壁涂黄漆 + 非加工外表面涂色）
工序 10：检（总检入库）
```

**核心教学要点**：
- ✅ **回转台多工位**：4 个面的孔系用一次装夹完成，4 个工位 = 旋转 0°/90°/180°/270°，**消除了 4 次重新装夹的累积误差**
- ✅ 工位 1 集成最多操作（4 类孔 + 攻丝）减少换刀
- ✅ 涂装在最前最后两次：底漆防锈 + 面漆装饰
- ✅ 时效紧跟铸造，消除内应力，避免后续机加工变形

## 案例 2：上壳体（6061 铝铸件）

**特殊点**：多种螺纹混合（M14×1 / M20×2 / M5 / M6 / M8 / M18 / M60×2）+ 多个斜孔/退刀槽/沉孔

```
工序 1-3：铸 → 时效 → 涂装
工序 4：粗铣上端面（夹下端，留 0.5mm 精铣余量）
工序 5：⭐ 数控铣（夹上端，四爪+回转台，12 个子工步）
  - 粗精铣下端面、钻扩 φ15、铣 φ58 孔留余量、钻 φ12 深 60
  - 铣 φ29/φ53 环形槽 + 腰形槽（±20°）+ φ22.5 外圆
  - 粗精铣 φ58H7 工艺定位孔
  - 铣 4×φ60.5 退刀槽（φ16×4 槽铣刀）
  - 铣 6.2×φ63.7 沟槽（φ16×4 槽铣刀）
  - 孔口/外圆倒角
  - 铣 M60×2-6H 螺纹（用螺纹刀）
工序 6：⭐ 数控铣（回转台 + 专用夹具，多种螺纹）
  工位 1：钻扩 M14×1 底孔 → 扩 M20×2 底孔 → 攻 M14×1 / M20×2 螺纹
  工位 2（180°）：钻扩 M20×2 底孔 + 攻丝
  工位 3（90°）：钻 2×φ7 通孔
工序 7：⭐ 数控铣（专用铣夹具，**最复杂工序**）
  - 精铣上端面、钻扩铰 φ16.5H8、铣 φ29 沉孔
  - 铣 2×φ19 退刀槽 + 20°倒角
  - 钻 φ3 直孔
  - **主轴旋转 30°，铣小平面 + 钻 φ3 斜孔**
  - 主轴垂直回正：攻 M18 / M8-6H / M5-6H 螺纹
工序 8：涂装
工序 9：检
```

**核心教学要点**：
- ✅ **多种螺纹规格在不同工序处理**：粗牙 M5/M6/M8 和细牙 M14×1/M20×2 分开（细牙公差严，单独控制）
- ✅ ⭐ **主轴旋转 30° 加工斜面/斜孔**：先铣斜小平面（防止钻头打滑） → 再钻斜孔
- ✅ ⭐ **退刀槽用槽铣刀（φ16×4 / φ16×2）**：避免普通立铣刀无法清根
- ✅ ⭐ **φ58H7 作为后续工序的工艺定位孔**（基准统一原则的实战）
- ✅ **环形槽 / 腰形槽都用 φ12 立铣刀**（同一刀完成多个特征，减少换刀）

## 案例 3：水泵叶轮（HT150 铸件）

**特殊点**：数控车 + 数控铣联用；叶片侧面加工；高精度铰孔（φ5H7、φ8H7、φ17）

```
工序 1-2：铸 → 时效
工序 3：车（夹 φ108 毛坯外圆，三爪，校正 3.5 尺寸左端面，粗车右端面留余量）
工序 4：⭐ 数控车（三爪，7 个子工步）
  - 粗车左端面 + 叶片左端面（尺寸 51、28.5 各留 1mm 余量）
  - 钻 φ15 → 粗车 φ35 孔留 1.5mm
  - 精车叶片端面（留 0.5mm）
  - 扩铰 φ17（车孔刀 + φ17 铰刀）
  - 半精+精车 φ35（深度至要求）
  - 割 1.6 槽（用 1.6 割槽刀）
工序 5：车（车心轴定位 φ17 孔，精车右端面 + φ75 沉孔 + 叶轮外圆 φ108 至要求）
工序 6：⭐ 数控铣（回转台 + 专用铣夹具，4 个工位）
  工位 1：铣 10×18 槽（**φ300 三面刃铣刀**）
  工位 2（回转台 90°）：再铣 10×18 槽
  工位 3（回转台 30°）：钻铰 φ5H7（**φ5H7 铰刀**）
  工位 4（回转台 90° + 主轴回转 90°）：
    - 2×φ8 处铣小平面（**φ8 立铣刀**）
    - 钻铰 φ8H7（**φ8H7 铰刀**）
工序 7：钳（修去毛刺）
工序 8：检
```

**核心教学要点**：
- ✅ ⭐ **车 / 数控车 / 数控铣分阶段**：粗车（普通车床效率高）→ 数控车（精度+效率）→ 数控铣（多工位）
- ✅ ⭐ **车心轴**：以加工过的精度孔（φ17）作为后续装夹基准（基准统一原则）
- ✅ ⭐ **三面刃铣刀 φ300 铣槽**（10×18）：直径远大于槽宽，效率极高（比立铣刀快 5-10 倍）
- ✅ ⭐ **铰孔到 H7**：φ5/φ8 这种小孔不能用镗，**钻 → 铰** 是唯一路径
- ✅ ⭐ **主轴 + 回转台双轴联动**：工位 4 是 90° 回转台 + 主轴 90° 回转，加工叶片侧面 φ8H7 孔

## 案例 4：刀盘（42CrMo + 淬火 HRC29-34）⭐⭐⭐ 必读

**特殊点**：合金钢 + 淬火 + 高精度（φ25H6 定位孔 + 5×R30 刀槽）

```
工序 1：⭐ 数控车（夹毛坯 φ108，三爪）
  - 粗车上端面 + φ60 外圆 + 圆锥面 + R5（各留 1.5mm 余量）
  - 钻 φ22 通孔 + 粗车 φ32 孔留 1.5mm
工序 2：数控车（夹 φ60 外圆调头）
  - 粗车下端面 + 粗车 φ42 孔留 1.5mm
工序 3：⭐ **热处理：淬火 HRC29-34**
工序 4：⭐ 数控车（**软爪**，夹 φ98 外圆）
  - 车上端面 + 准 φ60 外圆 + 圆锥面 + R5 + φ98 外圆
  - 车 φ25 / φ32 孔留 0.5mm
工序 5：数控车（软爪，调头）
  - 车下端面（总长 55 留 0.25mm）+ 精车 φ42 孔 + 车 6° 内锥面
工序 6：铣（粗铣键槽，10 留 0.5mm，6 深至 6.25mm）
工序 7：⭐ **磨**（软爪定位 φ98 外圆 + 下端面）
  - 磨 φ25H6 定位孔 + φ32 孔至要求 + 磨上端面
工序 8：⭐ **工具磨**（基准 φ25H6 孔）
  - 磨键槽宽度 10 至要求
工序 9：⭐ 数控铣（分度头 + 心轴）
  工位 1：**主轴沿工件轴线方向前倾 4°（图样 86°）**
    - 铣 5×R30 槽（**φ60 立铣刀**）
    - 钻、铣刀槽两定位面处小孔
  工位 2：**主轴向前倾 7°，分度头逆转 11°**
    - 粗精铣 5 个刀槽底平面
    - 钻攻 M2.5 螺孔（M2.5 丝锥）
工序 10：钳（修去毛刺 + 安装刀片，百分表校验）
工序 11：检
```

**核心教学要点 ⭐⭐⭐**：
- ✅ **完整淬火件工艺路线**：
  ```
  粗车×2 → 淬火 → 软爪精车×2 → 半精铣键槽 → 磨削（孔+面）→ 工具磨（键槽）→ 数控铣（刀槽）
  ```
- ✅ ⭐ **淬火前留余量**：粗车留 1.5mm（淬火前余量），淬火后软爪车留 0.5mm（精车前余量）+ 0.25mm（磨削前余量）
- ✅ ⭐ **软爪**：淬火件硬度高 + 工件已加工，**普通卡爪会划伤**——必须用软爪（铜/铝爪铸定）
- ✅ ⭐ **磨 vs 工具磨**：
  - 普通磨：磨内孔 / 平面（外圆磨床 / 平面磨床）
  - 工具磨：磨键槽 / 异形小特征（工具磨床 + 分度装置）
- ✅ ⭐ **分度头 + 主轴倾斜联用**：刀盘 5 个刀槽是 R30 弧形，分度头 5 等分（72° / 圈），主轴前倾 4°/7° 配合工件轴线斜角
- ✅ ⭐ **图样 86° = 主轴前倾 4°**（90°-86°=4°）；图样 79° = 主轴前倾 11°（90°-79°=11°）—— **角度换算很容易出错**

## 案例 5：尾座体（铸件，传统精密工艺）⭐⭐⭐

**特殊点**：长导轨 A/B 面 + φ17H6 高精度孔 + 圆柱度 0.004 + 平行度 0.005

```
工序 1-3：铸 → 时效 → 涂装
工序 4：划线（基准 16 上平面 + φ35 外圆，划各外表面加工线）
工序 5：⭐ **刨**（按线找正）
  - 粗刨 A 面、B 斜面、45°沟槽、10 台阶面
  - 精刨 A 面、B 斜面，达到 Ra3.2
工序 6：⭐⭐ **粗精刮研** A 面、B 斜面，**10-13 点/25×25**
工序 7：镗（专用镗夹具，基准 A 面 + B 面 + φ35 划线）
  - 粗半精铣 φ35 两端面（保证 80）
  - 粗半精精镗 φ17 孔（**留研磨量**）
工序 8：钻（专用钻夹具，基准 A/B 面 + φ35 端面）
  - 钻 φ14 + 平锪 φ26
  - 钻攻 M6 螺纹
工序 9：⭐⭐ **研磨** φ17H6（保证圆柱度 0.004，配研）
工序 10：⭐⭐ **配刮** A 面、B 斜面，**10-13 点/25×25，保证平行度 0.005**
工序 11：涂装（内壁涂黄漆 + 非加工面涂色）
工序 12：检
```

**核心教学要点 ⭐⭐⭐**：
- ✅ **刮研**（手工工艺）：用于**长平面 / 配合面 / 几何精度极高（平行度 0.005、平面度 0.005）**——磨削难做长零件，刮研更可控
  - **覆盖率指标**：10-13 点/25×25mm² = 高级精度（精密机床导轨标准）
  - 8 点/25×25 = 一般精度
  - 16+ 点/25×25 = 超精
- ✅ **研磨**（手工 / 半手工）：用于**小孔超精度（IT6 及以上 + 圆柱度 0.005 以下）**
  - 配研：用配套的研具（如轴磨配孔），**两件互为研磨**
- ✅ ⭐ **铸件长导轨经典工艺**：
  ```
  铸 → 时效 → 划线 → 粗刨 → 精刨 → 粗刮 → 精刮 → （配研）→ 配刮
  ```
  注意"配刮"在最后——**配刮不只是加工，更是装配**（配合相邻零件的刮研，所以在装配工序之后）
- ✅ **镗孔 → 研磨**：先镗到接近尺寸（留 0.01-0.02mm 研磨量），再研磨到最终公差

## 跨案例规则（**重要！**）

### 规则 A：基准选择的实战
| 案例 | 粗基准 | 第一精基准 | 后续基准 |
|---|---|---|---|
| 1 箱体 | R35 毛坯外圆 + φ30 凸台 | φ58 内孔 + 下端面 | 同上 |
| 2 上壳体 | 外形 + 下端面 | φ58 孔 + 下端面 + 后侧面 | 同上 |
| 3 叶轮 | φ108 毛坯外圆 | φ17 孔（**车心轴**） | 右端面 + φ17 + 叶片侧面 |
| 4 刀盘 | 毛坯 φ108 / φ60 外圆 | φ60 外圆 + 端面（淬前）→ φ98 外圆（淬后软爪） | φ32 孔 + 上端面 |
| 5 尾座 | 16 上平面 + φ35 | A 面 + B 面 | A/B 面 + φ35 |

**铁律**：**精基准一旦确定，必须贯穿后续所有工序**（基准统一）

### 规则 B：热处理时机标准链
```
铸件件: 铸 → 时效（粗加工前）→ 加工
                                    
锻件 / 调质钢: 锻 → 正火/退火 → 粗加工 → 调质 → 精加工

淬火件: 粗加工 → 淬火 → 软爪精加工 → 磨削
                                          
渗碳件: 粗加工 → 半精加工 → 渗碳 → 淬火回火 → 磨削
```

### 规则 C：精度要求 → 工艺映射（来自案例 4-5 实战）
| 精度等级 | 工艺路线 | 案例 |
|---|---|---|
| IT9-10 (Ra 3.2) | 钻 → 半精铰 / 粗镗 | 通用 |
| IT8 (Ra 1.6) | 钻 → 扩 → 铰 / 镗 | 案例 1 φ12 |
| IT7 (Ra 0.8) | 钻 → 扩 → 粗铰 → 精铰 / 粗镗 → 精镗 | 案例 2 φ16.5H8 |
| **IT7 (Ra 0.4)** | 钻 → 镗 → **磨**（淬火件）| 案例 4 φ32 |
| **IT6 (Ra 0.4)** | 钻 → 粗镗 → 精镗 → **磨** / **配研** | 案例 4 φ25H6, 案例 5 φ17H6 |
| **IT5 + 圆柱度 ≤ 0.005** | 钻 → 镗 → 磨 → **手工研磨/配研** | 案例 5 |

### 规则 D：表面质量 → 工艺（长平面/导轨）
| 长平面精度 | 工艺 | 覆盖率 |
|---|---|---|
| Ra 3.2 平面度 ≤ 0.05 | 粗铣 → 精铣 / 精刨 | — |
| Ra 1.6 平行度 ≤ 0.02 | 粗铣 → 半精铣 → 粗磨 → 精磨 | — |
| **Ra 0.8 平行度 ≤ 0.005** | 粗刨 → 精刨 → **粗刮 → 精刮** | **10-13 点/25×25** |
| Ra 0.4 平行度 ≤ 0.003 | 同上 + **配刮** | 16+ 点/25×25 |
